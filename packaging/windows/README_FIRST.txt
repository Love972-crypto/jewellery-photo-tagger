SUNAAR JEWELLERY TAGGER - WINDOWS
=================================

FIRST TIME
1. Extract the complete ZIP to a normal folder. Do not run it from inside the ZIP.
2. Double-click "Run Sunaar Jewellery Tagger.bat". Internet is not required for
   the complete offline package.
3. The app uses its own bundled official signed Python 3.14.6 runtime with tkinter.
   It never uses or changes another Python installation on the computer.
   If any private Python/Tk file is missing, a verified bundled repair archive
   restores it atomically inside this app folder.
   On first launch it installs the exact locked app dependencies from the bundled,
   checksum-verified offline wheel cache. It does not search the Internet.
   The full Office package already includes the verified OCR, U2Net, and BiRefNet models.
4. A "Sunaar Jewellery Tagger" desktop shortcut is created automatically.
5. Keep the black launcher window open while using the app.

AFTER FIRST SETUP
- The app, dependencies, OCR models, and background models work offline.
- Private Python and app dependencies stay inside the extracted app folder.
- Keep the black launcher window open while using the app.
- If the desktop icon is missing, run "Create Desktop Shortcut.bat" once.

IMPORTANT
- "Install or Repair Dependencies.bat" restores private Python/Tk and repairs setup if needed.
- The safe launcher avoids an unsigned custom EXE, which antivirus and Drive can falsely flag.
- Full-quality PNG output is unchanged from the canonical local app.
- Tags, disconnected jewellery pieces, chains, pearls, and thin details use preservation mode.
- The 20 KB ZIP is an optional extra export and never replaces full-quality images.
